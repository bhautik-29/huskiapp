Changing Font

font-family Americana Font use to as a heading and font-family Madera W01 Regular use to as a details text.this is downloaded fonts. To change the Font, you will need to Edit the Bottom Link with your Custom Font link in style.css file and change the body css font-family

If you plan to use a Self Hosted font. Here is an Example for using Self Hosted Font.

    @font-face {
          font-family: "Americana Font"; 
        src: url(../fonts/Americana-font/Americana-Font-Regular1.ttf) format("truetype");
      }
      
      @font-face {
          font-family: "Madera W01 Regular"; 
        src: url(../fonts/Madera-W01-Regular/Madera-W01-Regular.ttf) format("truetype");
      }

      :root {
        --white-color--: #ffffff;
        --black-color--: #000000;
        --light-pink--: #f8f8f8;
        --look-book-color--: #f4fdff;
        --gray--: #575757;
        --navlink-color--: #393939;
        --nav-background-color--: #f4f4f4;
        --section-heading--: #343434;
        --section-bgc-color--: #f3f3f3;
        --footer-backgroundcolor-black--: #090909;
        --font-family-heading--: "Americana Font";
        --font-family-details--: "Madera W01 Regular";
      }
